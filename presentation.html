Properties:
  • Input: any integer
  • Output: a n-bit integer
  • Deterministic
  • Unpredictable

One linked list to rule them all!

  • "Blocks" are added onto the "top" of the linked list (a.k.a. "chain") by reference to the hashed value of the previous block
  • But who was front?

Sir Hashalot*

  • Solution: add difficulty
    • Decree:
    Henceforth, all new blocks must get low-low-low-low (hashes).**
  • Allow only the lowest fraction of the possible n-bit hash outputs
  • Use n(onces)
  • Target: one block / 10 minutes
  • Now we have to cooperate
  • Offer a reward for inclusion
